# Polityka prywatności bota "NazwaBota"

## 1. Dane zbierane
Bot loguje zmiany konfiguracji wykonane przez administratorów serwera. 
Logi zawierają:
- ID użytkownika w formie **hashowanej z solą serwera**
- ID serwera i kanałów

## 2. Cel przetwarzania
- Zapewnienie bezpieczeństwa i przejrzystości w zarządzaniu konfiguracją bota
- Ułatwienie administracji i debugowania działania bota

## 3. Przechowywanie danych
- Dane są przechowywane lokalnie w pliku JSON
- Hashowanie ID użytkowników uniemożliwia ich powiązanie z osobą fizyczną
- Dostęp mają wyłącznie administratorzy serwera/bota

## 4. Udostępnianie danych
- Dane nie są udostępniane osobom trzecim i nie są wykorzystywane komercyjnie
