# Regulamin korzystania z bota "NazwaBota"

## 1. Użycie bota
- Bot może być używany tylko w celach zgodnych z prawem i regulaminem serwera
- Zabrania się używania bota do spamu, nadużyć lub obejścia zabezpieczeń

## 2. Ograniczenie odpowiedzialności
- Bot jest dostarczany „tak jak jest” (as-is)
- Twórca bota nie ponosi odpowiedzialności za szkody wynikłe z korzystania z bota

## 3. Dane użytkowników
- Bot może logować pewne dane (hashowane ID użytkownika, ID serwera i kanałów)
- Dane służą wyłącznie administracji bota i nie są udostępniane osobom trzecim

- Twórca bota zastrzega sobie prawo do zmiany regulaminu
